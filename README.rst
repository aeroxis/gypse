Getting Started
---------------

    pip install gypsy

Usage
-----

Gypsy comes with some sample data that you can clone and play with.

    git clone https://github.com/aeroxis/gypsy /tmp/gypsy
    cd /tmp/gypsy
    gypsy url-extractor ./samples/ --margins 5
