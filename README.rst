Getting Started
---------------

    pip install gypse

Usage
-----

Gypsy comes with some sample data that you can clone and play with.

    git clone https://github.com/aeroxis/gypse /tmp/gypsy
    cd /tmp/gypsy
    gypse url-extractor ./samples/ --margins 5
